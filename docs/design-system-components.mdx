## 最佳实践

### ✅ 推荐用法

1. **保持一致性**：在同一页面中使用相同的组件样式
2. **语义优先**：选择符合场景的语义变体（如 danger 用于删除操作）
3. **适度使用**：避免在同一页面堆砌过多演示组件
4. **响应式考虑**：在移动端测试组件显示效果

### ❌ 避免用法

1. 不要在正式文档中混合使用多种按钮变体
2. 不要硬编码颜色值，始终使用语义 Token
3. 不要在演示区放置过多内容（保持简洁）
4. 不要忽略可访问性（确保足够的对比度）
