# 从 STM32F407 迁移到 XX32F407

:::tip 本页摘要
STM32F407VGT6 → XX32F407VGT6 迁移指南。Cortex-M4 内核，最高 168MHz，带 FPU 和 DSP 指令，100% 兼容无需修改代码。
:::

## 兼容性总览

XX32F407 与 STM32F407 **100% 兼容**。

| 项目 | STM32F407 | XX32F407 | 兼容性 |
|------|-----------|----------|--------|
| 内核 | Cortex-M4 with FPU | Cortex-M4 with FPU | ✅ |
| 主频 | 168MHz | 168MHz | ✅ |
| Flash | 1MB | 1MB | ✅ |
| SRAM | 192KB | 192KB | ✅ |
