# 外设对照表

:::tip 本页摘要
STM32F103 与 XX32F103 外设逐项对照表，包含寄存器地址、功能特性、引脚复用等详细对比信息。
:::

## 外设总览

| 外设 | STM32F103 | XX32F103 | 兼容性 |
|------|-----------|----------|--------|
| GPIO | ✅ | ✅ | 100% |
| RCC | ✅ | ✅ | 100% |
| NVIC | ✅ | ✅ | 100% |
| USART | ✅ | ✅ | 100% |
| SPI | ✅ | ✅ | 100% |
| I2C | ✅ | ✅ | 100% |
| ADC | ✅ | ✅ | 100% |
| TIM | ✅ | ✅ | 100% |

| 特性 | STM32F103 | XX32F103 |
|------|-----------|----------|

| 每端口引脚 | 16 | 16 |
| 模式 | 8 种 | 8 种 |
| 中断 | ✅ | ✅ |
