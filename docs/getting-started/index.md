# 快速入门

:::tip 本页摘要
快速入门模块提供 XX32 MCU 开发的完整入门指南，从环境搭建到第一个 LED 闪烁项目，帮助您快速上手开发。
:::

欢迎开始 XX32 MCU 的开发之旅！

## 本章内容

### 1. [概览](/docs/getting-started/overview)

了解本模块的学习路径和前置知识。

### 2. [环境搭建](/docs/getting-started/environment-setup)

学习如何安装和配置开发环境：
- STM32CubeIDE 安装
- 驱动程序安装
- 项目模板导入

### 3. [Hello World](/docs/getting-started/hello-world)

创建您的第一个 XX32 MCU 项目：
- LED 闪烁项目
- GPIO 配置
- 编译和下载

### 4. [GPIO 入门](/docs/getting-started/gpio-blink)

深入学习 GPIO 的使用：
- 按键检测
- 中断处理
- 完整代码示例

## 开始学习
