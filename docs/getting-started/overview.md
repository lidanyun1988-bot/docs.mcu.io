# 快速入门概览

:::tip 本页摘要
快速入门模块将引导您从零开始学习 XX32 MCU 开发，涵盖环境搭建、Hello World 项目创建和 GPIO 基础使用，帮助您快速上手开发。
:::

欢迎开始 XX32 MCU 的开发之旅！本模块将帮助您快速搭建开发环境并完成第一个项目。

## 学习路径

### 1. 环境搭建

学习如何安装和配置开发所需的工具和软件：

- IDE 安装（Keil MDK / IAR / STM32CubeIDE）
- 工具链配置（编译器、调试器）
- 驱动程序安装
- 项目模板导入

<Link to="/docs/getting-started/environment-setup">开始环境搭建 →</Link>

### 2. Hello World

创建您的第一个 XX32 MCU 项目，实现 LED 闪烁：

- 创建新项目
- 配置时钟和 GPIO
- 编写第一个程序
- 编译和下载

<Link to="/docs/getting-started/hello-world">开始 Hello World →</Link>

### 3. GPIO 入门

深入学习 GPIO 的使用，包括输入和输出：

- GPIO 工作原理
- 输出模式控制 LED
- 输入模式检测按键
- 中断处理

<Link to="/docs/getting-started/gpio-blink">学习 GPIO →</Link>

## 前置知识

- 基础 C 语言编程能力
- 了解单片机基本概念
- 有任意 MCU 开发经验更佳

## 开发工具

- **推荐 IDE**: STM32CubeIDE（免费）或 Keil MDK
- **调试器**: ST-Link / J-Link / DAP-Link
- **串口工具**: PuTTY / SecureCRT / 串口助手
